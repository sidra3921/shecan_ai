# SheCan AI - App Overview

## 🎯 Application Summary

SheCan AI is a comprehensive Flutter application designed to empower women through freelancing, mentorship, and skill development. The app features a beautiful pink-themed UI and includes all the screens from your design mockup.

## 🚀 How to Run

1. Ensure Flutter is installed (SDK 3.8.1+)
2. Navigate to project directory: `cd e:\shecan_ai`
3. Install dependencies: `flutter pub get`
4. Run the app:
   - Mobile: `flutter run`
   - Windows: `flutter run -d windows`
   - Web: `flutter run -d chrome`
